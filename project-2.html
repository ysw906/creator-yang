과학 디지털 콘텐츠 개발

영상, 인터랙티브 콘텐츠, 디지털 교과서를 학습 흐름에 맞춰 설계한 콘텐츠 개발 프로젝트

디지털 콘텐츠영상 기획 UI/UX인터랙티브

비상교육 디지털 교과서 콘텐츠 화면

프로젝트 개요

  • 영상 교재: 과학 개념 설명 영상 기획 및 스크립트 작성
  • 인터랙티브 콘텐츠: 개념 탐구형 웹 콘텐츠 기획
  • 디지털 교과서: AI 학습 추천 시스템, 콘텐츠 연동 설계

핵심 기획

1. 영상 콘텐츠 기획

  • 학습 흐름을 유지하는 구성 설계
  • 스크립트 → 씬 구성 → 연출 기획

2. 인터랙티브 UI/UX 설계

  • 탐구 순서에 맞는 인터랙션 흐름 설계
  • 개념 이해를 돕는 시각 요소 기획

3. 디지털 교과서 콘텐츠 연동

  • AI 추천 시스템과 콘텐츠 연결 구조 설계
  • 학습 단계별 콘텐츠 제공 흐름 기획

나의 역할

  • 디지털 콘텐츠 기획 및 구조 설계
  • 영상 스크립트 작성
  • UI/UX 방향성 제안

핵심 역량

  • 멀티미디어 콘텐츠 기획
  • 사용자 경험(UX) 설계
  • 학습 흐름 설계
